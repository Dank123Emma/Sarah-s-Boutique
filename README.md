# Sarah's Electronics - E-commerce Website

## Overview
Sarah's Electronics is a fully responsive, multi-page e-commerce website built with **HTML, CSS, and JavaScript**. Users can browse products, add them to the cart, create accounts, log in, and proceed to checkout. The site is designed as a **frontend-only application**, using `localStorage` to simulate account management and cart functionality.

## Features
- Multi-page structure:
  - Home
  - Shop
  - Product Page
  - Cart
  - Checkout
  - Account (Login/Signup)
- Responsive design for desktop, tablet, and mobile
- User authentication (login/signup)
- Add to cart and remove items
- Proceed to checkout
- Footer with About Us, Contact Us, Policies
- Modern UI with hover effects and card layouts
- Fully separated HTML, CSS, and JS files per page
- LocalStorage-based cart and user management

## Technologies Used
- HTML5
- CSS3 (Responsive, Flexbox, Grid)
- JavaScript (Vanilla JS)
- LocalStorage for session simulation

## Installation / Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/sarahs-electronics.git
   ```
2. Open `index.html` in your browser to start exploring.
3. Navigate through pages using the header menu.
4. Create an account or log in to add products to the cart and checkout.
